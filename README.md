# hw1e14

## inital setup

- npm install
- npm install hexo-cli -g
- git clone https://github.com/iissnan/hexo-theme-next themes/next
- hexo server

## deploy
- hexo new
- - hexo generate --deploy

## usage

- hexo new
- hexo list post
- hexo publish [layout] <title>
- hexo deploy [layout] <title>
- hexo generate --deploy
- hexo deploy --generate
